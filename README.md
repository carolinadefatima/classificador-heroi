# Classificador de Nível de Herói (Lógica de Negócios)

Projeto criado como parte do desafio do curso **"Ri Happy - Front-end do Zero"** (DIO), focado em aplicar **Lógica de Programação** para processamento de dados e tomada de decisão.

## Objetivo Técnico

O objetivo central desta aplicação foi desenvolver um **algoritmo robusto** capaz de processar uma variável de entrada (XP) e classificá-la em diferentes categorias de saída (Nível).

Este exercício simula a lógica de **regras de negócio**, onde a aplicação precisa tomar decisões complexas baseadas em dados de entrada.

## Tecnologias e Implementação

* **JavaScript:** Principal linguagem utilizada para implementar o **algoritmo de classificação**.
* **Estruturas Condicionais:** Uso de `if/else if/else` (ou `switch`) para cobrir todas as faixas de XP e garantir que o *input* receba o *output* correto.
* **Manipulação de DOM e Eventos:** Utilizado para capturar os dados do usuário e renderizar o resultado na interface.

## Níveis de Classificação (Regras de Negócio)

| Faixa de XP | Nível |
|:-------------:|:-------------:|
| Menor que 1000 | Ferro |
| 1001 - 2000 | Bronze |
| 2001 - 5000 | Prata |
| 5001 - 7000 | Ouro |
| 7001 - 8000 | Platina |
| 8001 - 9000 | Ascendente |
| 9001 - 10000 | Imortal |
| Maior que 10000 | Radiante |
